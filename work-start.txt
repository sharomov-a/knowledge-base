Разворачиваем локальную копию:

    Берем бэкапы файлов на гейтсервере
    Там же берем последний дамп БД
    Создаем виртуальный хост
    Файлы распаковываем, дамп импортируем в заранее созданную пустую БД
    Меняем реквизиты подключения в /bitrix/.settings.php и /bitrix/php_interface/dbconn.php
    Проверяем работоспособность сайта

Настраиваем Git:

    Создаем локально пустой репозиторий над корнем сайта: git init
    Добавляем удаленный репозиторий из GitLab: git remote add origin адрес_репозитория
    Обновляем, переключаемся на мастер: git fetch origin, git checkout -f -b master origin/master
    Отсюда уже можно создавать ветки под задачи через git checkout -b имя_ветки
